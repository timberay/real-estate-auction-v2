# Decision: T1.4(a)-full 인근 낙찰가 통계 인프라 — DEFERRED

**Date**: 2026-05-15
**Decider**: 사용자 명시 결정 (ROI 재검토 후 lite 옵션 선택)
**Status**: DEFERRED — 사용자 수요 누적 시 재검토
**Related**: 마스터 TODO T1.4(a), W2-1 / C22 / E-24, audit `docs/audits/2026-05-09-ux-audit-expert.md` E-24

## Context

T1.4(a)는 audit E-24 ("베테랑은 인근 낙찰가 통계 + 회차별 저감률 + 경쟁률 셋을 종합")의
(a) 부분으로, profit_calculator 에 "이 동네 비슷한 매물 매각가율"을 표시하기 위해
신규 `AuctionResult` 모델 + 스크래퍼 + 집계 서비스를 빌드하는 작업이었다.

원래 설계 (PR 1 + PR 2 분리):
- PR 1: `AuctionResult` 모델 + `CourtAuction::SoldResultSearchClient` + `AuctionResultsScrapeJob`
- PR 2: `NearbyAuctionStatsService` + profit_calculator 카드

ROI 재검토에서 다음 사실이 드러남:

1. **데이터 규모의 비대칭**: 전국 주거용 경매 매각 ~3,500~5,000건/년, 행정동 ~3,500개.
   유의미한 dong-level 표본 (≥10건/년)은 ~80~150개 동 (주로 강남3구·송파·분당·해운대 등).
   80% 매물에서 "데이터 부족" fallback 발생 → UX 의미 약함.
2. **대체재의 존재**: 디스코·옥션원·지지옥션·지지옥션 등 상업 서비스와 법원경매정보 공식
   매각통계 페이지가 이미 매각가율 통계를 제공. 베테랑은 이미 외부 도구 병용 중.
3. **다른 Theme 1 항목과의 패턴 불일치**: T1.1~T1.5 의 다른 항목 (취득세/양도세/소액임차인/
   DSR/차회저감) 은 모두 closed-form 계산. T1.4(a) 만 외부 데이터 파이프라인 의존 — 
   유지보수 비용(스크래퍼 깨짐·rate limit·표본 부족 컴플레인) 비대칭.

## Options

| Option | Cost | Value | 위험 |
|---|---|---|---|
| **A. 외부 링크 lite** (NearbyAuctionLinksComponent → courtauction.go.kr) | XS (~2h, 1 PR) | 베테랑 외부 도구 병용 가정에서 ~60~70% | 외부 의존 |
| **B. 완전 defer** | 0 | 0 즉시, 검증된 수요 확보 후 | 응답 지연 |
| **C. sigungu-only narrow** | M (1.5 PR) | ~50% (dong 의미 약함) | 인프라 부담 |
| **D. 원래 설계 full** | L (2 PR) | 표본 충분 지역 100%, 부족 지역 0 | 인프라+유지보수 |

## Decision: A (lite) 실행 + D (full) DEFER

**근거**:

1. **"숫자 없음 < 외부 링크 < 잘못된 평균"** — 표본 부족 지역에서 잘못된 평균을 보여주는
   것보다 권위 있는 공식 출처 링크가 베테랑 신뢰에 더 안전.
2. **Reversible** — lite 구현이 끝난 후에도 수요가 누적되면 언제든 full 인프라 도입 가능.
   반대 방향(full → lite 축소)은 매몰비용 발생.
3. **다른 Theme 1 closed-form 항목과 패턴 일치** — 외부 데이터 파이프라인 도입의 명분이
   사용자 수요로 검증된 시점까지 미룸.

## Re-evaluation Trigger

다음 중 하나라도 발생하면 T1.4(a)-full 재검토:

- 사용자가 "본 도구 안에서 인근 통계 보고 싶다"고 명시적으로 ≥3건 누적
- 베테랑 retention 분석에서 "외부 도구 병용 단절"이 이탈 원인으로 식별
- 외부 링크 도착률(클릭) 데이터가 충분히 높아 in-house 통계 노출이 컨버전 개선에 직결

재검토 시 우선 옵션 C (sigungu-only narrow) 부터 검토 — 표본 안정성 확보된 시군구 단위
출발이 가장 작은 인프라로 가장 큰 가치를 회수.
