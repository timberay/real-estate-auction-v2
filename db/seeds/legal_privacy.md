# 개인정보처리방침

**TIMBERAY**(개인사업자, 이하 "회사")는 부동산 경매 분석 서비스 "Real Estate Auction"(이하 "서비스")을 운영함에 있어 「개인정보 보호법」 및 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」을 준수하기 위하여 다음과 같이 개인정보 처리방침을 수립·공개합니다.

## 1. 수집하는 개인정보 항목

회사는 다음의 개인정보를 수집합니다.

### 가. SNS 간편 로그인을 통한 수집 (가입 필수)

| 항목 | Google OAuth | Naver OAuth | Kakao OAuth |
|------|--------------|-------------|-------------|
| 이메일 주소 | ✅ | ✅ | ✅ |
| 닉네임(표시 이름) | ✅ | ✅ | ✅ |
| 프로필 이미지 URL | ✅ | ✅ | ✅ |
| OAuth 공급자별 고유 식별자(UID) | ✅ | ✅ | ✅ |

### 나. 서비스 이용 과정에서 자동 생성 또는 이용자가 입력하는 정보

- 관심 매물 즐겨찾기·메모·태그
- AI 분석 요청 시 이용자가 입력한 프롬프트 텍스트 및 첨부 파일(매각물건명세서·등기부등본 등)
- 분석 결과 보고서 및 권리분석 편집 내용
- 예산 설정(BudgetSetting), DSR 계산 입력값
- 서비스 이용 기록, 접속 IP, 접속 로그, 쿠키, User-Agent

## 2. 수집·이용 목적

회사는 수집한 개인정보를 다음의 목적에 한하여 이용합니다.

- 회원 인증 및 SNS 간편 로그인 식별
- 이용자별 분석 기록·즐겨찾기·예산 설정의 저장 및 복원
- AI 분석 결과의 사용자 귀속 및 이력 관리
- 부정 이용 탐지 및 서비스 남용 방지(Rack::Attack 등 보안 모듈 운영)
- 서비스 개선을 위한 통계 분석(개별 식별이 되지 않는 형태로만)
- 법령상 의무 이행 및 분쟁 발생 시 분쟁 해결 자료

회사는 위 목적 외의 용도로는 개인정보를 이용하지 않으며, 목적이 변경되는 경우 별도의 동의를 받습니다.

## 4. 외부 LLM API 제공사

본 서비스는 AI 분석 기능 제공을 위해 다음 외부 LLM API 제공사에 이용자가 입력한 데이터(프롬프트 텍스트·첨부 파일 포함)를 전송합니다.

| 제공사 | 처리 국가 | 이전되는 항목 | 이전 목적 | 보유 기간 |
|--------|-----------|---------------|-----------|-----------|
| **Anthropic, PBC** (Claude API) | 미국 | 이용자 프롬프트, 첨부 파일, 분석 컨텍스트 | AI 권리분석·요약 결과 생성 | 제공사 정책에 따름(원칙적으로 30일 이내 파기) |
| **OpenAI, L.L.C.** (GPT API) | 미국 | 이용자 프롬프트, 첨부 파일, 분석 컨텍스트 | AI 권리분석·요약 결과 생성 | 제공사 정책에 따름(원칙적으로 30일 이내 파기, API 이용 시 학습 비활용) |
| **Google LLC** (Gemini API) | 미국 | 이용자 프롬프트, 첨부 파일, 분석 컨텍스트 | AI 권리분석·요약 결과 생성 | 제공사 정책에 따름 |

각 제공사의 데이터 처리 정책은 다음에서 확인하실 수 있습니다.

- Anthropic: https://www.anthropic.com/legal/privacy
- OpenAI: https://openai.com/policies/privacy-policy
- Google: https://policies.google.com/privacy

이용자가 외부 LLM API로의 데이터 이전에 동의하지 않을 경우 AI 분석 기능을 이용할 수 없으며, 동의 거부 시에도 서비스의 비-AI 기능(매물 조회·세금 계산기 등)은 이용 가능합니다.

## 5. 정보주체의 권리

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

- **열람**: 본인의 개인정보 처리 현황 및 보유 항목 조회
- **정정·삭제**: 잘못된 정보의 수정 또는 삭제 요청
- **처리정지**: 동의 철회 또는 처리 중단 요청
- **회원 탈퇴**: 서비스 내 "회원 탈퇴" 메뉴를 통한 즉시 탈퇴 및 개인정보 일괄 파기

권리 행사는 서비스 내 설정 메뉴 또는 개인정보 보호책임자 연락처(아래 9항)를 통하여 가능하며, 회사는 요청을 받은 날로부터 **10일 이내**에 처리 결과를 통지합니다.

## 6. 파기 절차

회원 탈퇴 또는 보유 기간 경과 시 회사는 다음 절차에 따라 개인정보를 파기합니다.

- **전자적 파일 형태**: 데이터베이스에서 즉시 삭제(soft-delete 후 7일 내 hard-delete 배치 처리). 백업 파일은 백업 주기에 따라 다음 백업 갱신 시 자동 덮어쓰기.
- **출력물 형태**: 해당 없음(본 서비스는 별도의 종이 출력물을 보관하지 않음).

## 7. 개인정보의 안전성 확보 조치

회사는 다음의 기술적·관리적 조치를 통하여 개인정보의 안전성을 확보하고 있습니다.

- HTTPS(TLS 1.2 이상) 적용을 통한 전송 구간 암호화
- OAuth 토큰 및 세션 정보의 암호화 저장
- 접근 권한 최소화 원칙 및 관리자 인증
- Rails Active Record 매개변수 바인딩을 통한 SQL 인젝션 방지
- Content Security Policy(CSP) 및 Rack::Attack을 통한 외부 공격 차단
- 정기적 보안 패치 및 의존성 취약점 점검
